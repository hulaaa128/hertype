# 《语言恶女 Wordslut》机制框架 · HerType 识别升级参考

> 来源：Amanda Montell《Wordslut: A Feminist Guide to Taking Back the English Language》中文译本《语言恶女》。
> 用途：把 HerType 从「固定词库匹配」升级为「按语言机制识别」的理论依据与 prompt 素材。
> 核心立场（可写进产品理念）：语言学家 Deborah Cameron 指出，英语的 DNA 里并不含性别偏见，是**使用方式**在不断再生产歧视——所以按「机制 + 不对称」判定，比死记词表更贴合本书原理。现有六分类词库只是这套机制的「已知实例」。

---

## 一、性别侮辱 / 贬低的七类底层机制（AI 判定维度）

### 1. 语义贬降（semantic derogation）
女性相关词随时间「词义转贬 pejoration」，对应的男性版却保持中性或转褒。
- **判定**：同一个词，是否只在指向女性时才染上淫秽 / 贬义？
- **英文例**：master↔mistress（男主人保权威，女主人=情妇）、sir↔madam（madam→鸨母）、governor↔hussy（husewif 主妇→贱妇）、buddy↔sissy、tart/slut/bitch 均由中性沦为骂女。
- **中文对应**：「先生」保值 vs 「小姐 / 大姐 / 妇人」贬值。

### 2. 把女性性化 / 拿性经验当武器
处女—荡妇二元论。女性欲望无论多寡都可耻（丰富=荡妇，克制=假正经 / 冰雪公主）。
- **判定**：该词是否以「性行为数量」给女性定罪？
- **例**：slut / whore / ho / skank / 破鞋 / 绿茶婊 / 公交车。
- （对应现词库 `sexual`）

### 3. 把女性非人化——比作动物 / 食物
贬到与被猎捕、被驯服、被吃之物同级。
- **判定**：是否用可食用物 / 牲畜隐喻女性身体？
- **动物**：bitch / heifer / 母牛 / hellpig；**食物**：tart / cupcake / peach / 嫩肉 / 鲜肉（书中点破：女性总被喻为「外硬里软、多汁、可切成几块、单人份易到手」之物）。
- ⚠️ 现词库未单列此类，是候选新增机制。

### 4. 用女性气质本身当侮辱（男性侮辱多来自「像女人」）
大部分骂男人的话，本质是指其「女性化」。
- **判定**：侮辱力是否来自「你没做好男人 / 像女人」？
- **例**：pussy / sissy / wimp / pansy / 「别跟个女人似的」/ 娘炮 / 娘娘腔。
- （对应现词库 `feminine`）

### 5. 外貌规训 / 能动性污名化
批评女性「不够甜美温顺、要太多权力」；同一强势行为，男性叫 leader，女性叫 bossy。
- **判定**：把女性的野心 / 情绪 / 强势贴上负面标签，男性做同样的事却是褒义？
- **例**：bossy / nasty / nag / hysterical（歇斯底里）/ hormonal / overemotional / 专横 / 母老虎 / 女强人；girlboss / SHE-EO（反而暗示 CEO 默认男）。
- （对应现词库 `appearance` + `rivalry`）

### 6. 阳性泛指 / 默认男性
person 默认=中产白男；女性职业要加「女」前缀。
- **判定**：是否需要额外加性别标记才成立？
- **例**：女医生 / 女科学家 / 女司机（而护士 / 保姆默认女）；manbun / guyliner（暗示化妆是女性专属，男性用要「男性化重新包装」）。
- ⚠️ 现词库未单列此类，是候选新增机制。

### 7. 称谓不对称 / 降格亲昵
对女性用甜心 / 小姑娘 / 宝贝 / 以发色称呼，对男性直呼其名——用亲昵标记从属地位。
- **判定**：称呼是否在无端降格、暗示对方低你一等？
- **例**：sweetheart / 甜心 / 小姑娘 / 宝贝 / 「金发妹」。书中指出同机制也用于对黑人司机说「我的伙计」。
- ⚠️ 现词库未单列此类，是候选新增机制。

## 二、可操作的识别信号（让 AI 按原理判断，而非查词典）

- **镜像 / 对等测试（mirror test）**：该词有没有语义对等的男版？若 manslut 需加前缀、slut 无需 → 不对称=父权词。**这是 HerType 镜像模式的核心。**
- **同一行为男女用词不对称**：滥交男=stud / 唐璜（褒），女=荡妇（贬）；说脏话男=正常，女=「不淑女」。
- **成就归因于身体 / 外貌**：书中 Yvonne Brill 讣告案——火箭科学家的讣告先写「拿手菜炖牛肉、好妈妈」，把女性成就压回家庭 / 身体。（对应现词库 `merit`）
- **female 作名词**：书中语料证明 female 当名词几乎只出现在贬损语境（暗示「生理缺陷」），woman 更中性。
- **规训性动机**：这句话是否在「惩罚对方不符合其性别刻板形象」？
- **声音 / 表达规训**（第八章）：批评女性 uptalk（句尾上扬）、vocal fry（气泡音）、爱说 like、过度道歉——背后用「中年白男」当无形标准。

## 三、镜像模式的理论支撑（可引用进 UI 文案 / 批判卡片）

书中反复用「性别对调」揭露不对称，可直接引用：

- **第五章 Robin Lakoff**：「每说一次『女总统』，都在强化只有男人才能当统帅」——把「女×」前缀对调即暴露歧视。
- **第一章反证**：若采用女性视角，「称好友的词会是 sissy 而非 buddy，pussy 会指『骁勇女王』」——对调后语义荒谬，即证明原词偏见。
- **第六章 · 喜剧演员 Peter White 黄金法则**：「别在街上对女人说任何你不想在监狱里听到男人对你说的话」——即对调施受双方来测冒犯度。
- **第七章 · 脏话不对称**：「suck my dick」能当去性化玩笑，「eat my pussy」不行，证明脏话力量默认男性视角；作者提议造 clit / suck my clit / holy clit 来翻转视角。

## 四、对 HerType 的落地建议

1. **镜像模式 AI prompt（`ai.js` MIRROR_PROMPT）**：从「找性别化表达 + 给男版对调」升级为「按上述 7 类机制原理判断」，每处命中额外输出 `mechanism`（命中哪类机制）与 `asymmetry`（不对称点：同一行为男女用词差异）。让 AI 抓词库外的表达，靠原理而非「像不像已知词」。
2. **词库分类（`lexicon.js` CATEGORIES）**：现有 6 类（maternal / sexual / feminine / rivalry / appearance / merit）覆盖机制 2/4/5 与 merit 信号。候选新增：**非人化（动物/食物）**、**阳性默认/称谓降格**、**语义贬降**。是否新增取决于是否要在 UI 上单独成色。
3. **学习模式保持纯本地**：学习模式不联网是其隐私卖点（「完全本地运行、不上传任何输入」）。机制识别若要进学习模式需联网，会破坏该承诺——建议机制识别只放在镜像 / 输出这两个已联网的模式里。
